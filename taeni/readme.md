Taeni
=======

Multi-tabbed terminal emulator based on VTE

How to install?
````
cd vala-apps/taeni
mkdir build; cd build
cmake ..
make; make install
gtk-update-icon-cache /usr/share/icons/hicolor
glib-compile-schemas /usr/share/glib-2.0/schemas
````
Dependencies:
````
gtk+-3.0 >= 3.22
vte-2.91
````
